📝 Test Final: Bases de Datos y MySQL Workbench

Pon a prueba tu conocimiento sobre bases de datos y MySQL Workbench con estas preguntas. Al final, recibirás tu calificación.

Pregunta 1

¿Cuál es el comando para crear una base de datos en MySQL?

Respuesta incorrecta. El comando correcto es CREATE DATABASE.

Pregunta 2

¿Qué tipo de dato se utiliza para almacenar cadenas de texto de longitud variable en MySQL?

Respuesta incorrecta. El tipo de dato correcto es VARCHAR.

Pregunta 3

¿Cuál es el comando para crear una tabla en MySQL?

Respuesta incorrecta. El comando correcto es CREATE TABLE.

Pregunta 4

¿Qué comando SQL se usa para insertar datos en una tabla?

Respuesta incorrecta. El comando correcto es INSERT INTO.

Pregunta 5

¿Cuál es el comando SQL para seleccionar todos los registros de una tabla?

Respuesta incorrecta. El comando correcto es SELECT * FROM.

Pregunta 6

¿Qué comando se utiliza para eliminar un registro de una tabla en MySQL?

Respuesta incorrecta. El comando correcto es DELETE.

Pregunta 7

¿Qué comando se utiliza para modificar un valor en una tabla en MySQL?

Respuesta incorrecta. El comando correcto es UPDATE.

Pregunta 8

¿Cuál es el comando para crear un índice en MySQL?

Respuesta incorrecta. El comando correcto es CREATE INDEX.

Pregunta 9

¿Qué tipo de JOIN devuelve todas las filas de la tabla de la izquierda y las filas coincidentes de la tabla de la derecha?

Respuesta incorrecta. El tipo de JOIN correcto es LEFT JOIN.

Pregunta 10

¿Cuál es el comando para establecer una clave primaria en una tabla en MySQL?

Respuesta incorrecta. El comando correcto es PRIMARY KEY.
