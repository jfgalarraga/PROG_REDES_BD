Ejemplos Access + VBA

Descripción: Una fruteria necesita registrar sus ventas para mantener un control preciso. Para esto se crea un formulario que permita ingresar datos como el producto vendido, la cantidad, el precio unitario del producto.

Solución: Se realiza una serie de pasos para este ejemplo:

Primer Paso: Verificar la base de datos de nuestras ventas

Segundo Paso: Procedemos a realizar la creación del formulario y insertamos el boton de agregar que nos permitira registrar la nueva venta

Tercer Paso: Accedemos a la parte de Vista Formulario e ingresamos el nuevo producto

Cuarto Paso: Regresamos a la tabla de ventas de nuestra fruteria y vemos el nuevo producto agregado

Solución: Se realiza una serie de pasos para este ejemplo:

Primer Paso: Verificar la base de datos de los empleados que tenemos

Segundo Paso: Procedemos a realizar la creación del formulario y insertamos el botón de leer productos que nos visualizar los datos de los productos

Tercer Paso: Accedemos a la parte de Vista Formulario e accionamos el boton de leer productos

Cuarto Paso: Observamos que estamos en la tabla productos
