Reto: Corregir Errores en la Base de Datos

En este reto, tu tarea es corregir los errores en una base de datos de clientes. Algunas filas tienen datos erróneos que debes corregir para pasar al siguiente reto. Si corriges más del 75% de los errores, podrás avanzar.

Instrucciones para corregir los datos

Para corregir las filas de datos, por favor sigue estas instrucciones:

  • ID: Debe ser un número, sin letras ni caracteres especiales.
  • Nombre y Apellido: Deben ser solo letras (sin números ni caracteres especiales), con espacios entre las palabras si es necesario.
  • Email: Debe seguir la estructura nombre.apellido@dominio.extension. Por ejemplo, juan.perez@gmail.com.
  • Fecha: Debe estar en formato YYYY-MM-DD (por ejemplo, 2024-01-25).
  • Edad: Debe ser un número de dos dígitos (por ejemplo, 25).
