Juego 2: Identificar relaciones de una Base de Datos

En este juego, tu objetivo es observar los datos de las tablas y hacer las relaciones correctas entre ellas. Tienes 3 minutos para analizar las tablas y responder las preguntas. ¡Buena suerte!

Contexto de la base de datos:

Esta base de datos simula un sistema de ventas. Tenemos cuatro tablas principales:

  • Clientes: Información sobre los clientes.
  • Pedidos: Registros de los pedidos realizados por los clientes.
  • Productos: Detalles de los productos disponibles.
  • Categorías: Información sobre las categorías a las que pertenecen los productos.
Analiza los datos y selecciona las relaciones correctas entre las tablas.
