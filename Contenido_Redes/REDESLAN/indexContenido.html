¿Qué es una red LAN?

Una red LAN (Local Area Network) es una red de comunicación de datos que conecta dispositivos dentro de un área geográfica limitada, como una casa, una oficina o un campus universitario.



Características de una Red LAN

Alcance limitado a una pequeña área geográfica, como una casa, oficina o campus.

Alta velocidad de transmisión de datos, normalmente en el rango de 1 Gbps o superior.

Conexión de dispositivos a través de cables o de forma inalámbrica (Wi-Fi).

Facilidad para compartir recursos como impresoras, archivos y conexiones a Internet.

Requiere de un equipo centralizado como un switch o router para la interconexión de los dispositivos.

Normalmente no requiere de un proveedor de servicios externo.

Componentes de una Red LAN

Router 1

Router de alta calidad para mejorar la conectividad Wi-Fi en grandes áreas.

Cable-Fibra Optica

Medio de transmisión que utiliza hilos de vidrio o plástico para enviar datos como pulsos de luz.

Servidor 1

Es una computadora o sistema que ofrece servicios, recursos o datos a otras computadoras (clientes) en una red.

Firewall 1

Es un dispositivo de seguridad de red que controla el tráfico de entrada y salida según reglas de seguridad preestablecidas.

Cable-Ethernet

Es un tipo de cable utilizado para conectar dispositivos a una red mediante cables de cobre

Servidor 2

Puede alojar sitios web, bases de datos, o aplicaciones, y generalmente está optimizado para manejar grandes volúmenes de información y proporcionar acceso a múltiples usuarios de forma simultánea.

Switch

Es un dispositivo de red que conecta múltiples dispositivos dentro de una red local (LAN)

Router 2

Se encarga de dirigir el tráfico de datos entre diferentes dispositivos en una red y entre tu red local e internet, eligiendo la mejor ruta para los datos.

Cable-Ethernet

Transmite datos a través de señales eléctricas. Los cables más comunes son el Cat5e y Cat6.

Firewall 2

Es un dispositivo o software que ayuda a evitar que personas no autorizadas accedan a tus sistemas, actuando como un guardia que examina el tráfico de red y decide si es seguro o no.

Computadora

Dispositivo electrónico que procesa, almacena y recupera datos. Se utiliza para realizar tareas como navegación en la web, procesamiento de texto, cálculos y ejecución de programas.

Firewall 3

Actúa como una barrera entre una red interna (como la de una empresa) y el internet, protegiendo la red interna de amenazas externas.

Guía para instalar Cisco Packet Tracer

Descubre cómo descargar, instalar y configurar Cisco Packet Tracer paso a paso en tu sistema operativo. ¡Comencemos!



I. Crear una cuenta en Cisco Networking Academy (NetAcad):

  1. Visita Cisco NetAcad.
  2. Haz clic en "Sign Up" o "Registrarse".
  3. Completa el formulario con tus datos personales.
  4. Acepta los términos y condiciones y verifica tu correo.

II. Descargar Cisco Packet Tracer

  1. Inicia sesión en Cisco NetAcad.
  2. Navega a "Recursos" y selecciona "Packet Tracer".
  3. Descarga la versión adecuada para tu sistema operativo (Windows, macOS, Linux).



  1. Descargue la versión de Packet Tracer que necesita.
    2. Packet Tracer 8.2.2 para Windows de 64 bits
  2. Inicie el programa de instalación de Packet Tracer.
  3. Inicie Cisco Packet Tracer seleccionando el ícono apropiado.
  4. Cuando se le solicite, haga clic en el botón verde Habilidades para todos para autenticarse.
  5. Se iniciará Cisco Packet Tracer y estará listo para explorar sus funciones.
