Conceptos Básicos

Los algoritmos realizan entre otras cosas, operaciones simples como lectura y escritura de datos, operaciones aritmeticas y comparaciones. También incluye estructuras de contol como condicionales y bucles, que de acuerdo a una condición se ejecutan cero o más veces las intrucciones.

Operaciones Básicas

Los algoritmos suelen realizar operaciones simples como:

1. Asignación

Es un operador unario que destruye la información existente de una variable.

Ejemplo:

dato = 5;

2. Lectura de datos

Obtener datos de una fuente, como el usuario o un archivo.

Ejemplo:

escribir("Ingresa un numero? ");

leer ( dato );

3. Escritura de datos

Mostrar resultados al usuario o guardar en un archivo.

Ejemplo:

escribir("El resultado es:", resultado);

4. Operaciones aritméticas

Son operadores binarios tales como: sumar, restar, multiplicar, dividir, calcular el módulo (resto de una división).

Ejemplo:

resultado = 5 + 3
resultado = 7 - 2
resultado = 4 * 6
resultado = 8 / 2
resultado = 9 % 4

5. Operadores racionales

Son operadores que se usan para formar condiciones simples. Estos operadores devuelve verdadero o falso .

Ejemplo:

resultado = (5 == 5) # Igualdad
resultado = (5 != 3) # Desigualdad
resultado = (7 > 3) # Mayor que
resultado = (3 < 7) # Menor que
resultado = (7 >= 5) # Mayor o igual que
resultado = (5 <= 7) # Menor o igual que

6. Conectores lógicos

Son operadores que unen dos o más condiciones simples para formar expresiones lógicas más complejas. Estos operadores devuelven un valor booleano (verdadero o falso).

Operador AND

El operador AND devuelve 1 (verdadero) solo si ambas condiciones son verdaderas. En cualquier otro caso, el resultado es 0 (falso).

Operador OR

El operador OR devuelve 1 (verdadero) si al menos una de las condiciones es verdadera. Solo será 0 (falso) si ambas condiciones son falsas.

Ejemplo:

resultado = (5 > 3) AND (8 < 10) # Devuelve verdadero si ambas condiciones son verdaderas
resultado = (7 < 5) OR (10 == 10) # Devuelve verdadero si al menos una condición es verdadera

    Variables y Tipos de Datos

    Las variables son contenedores que almacenan información, como números o texto. Los tipos de datos definen el tipo de valor que una variable puede almacenar.

    Tipos de datos:

    1. Enteros (int)

    Los enteros representan números enteros sin decimales. Pueden ser positivos o negativos.

    Ejemplo:

    numero_entero = 10

    2. Números de punto flotante (float)

    Los números de punto flotante representan números con decimales.

    Ejemplo:

    numero_flotante = 3.14

    3. Cadenas de texto (str)

    Las cadenas de texto representan secuencias de caracteres encerradas entre comillas simples o dobles.

    Ejemplo:

    mensaje = "¡Hola, mundo!"

    4. Booleanos (bool)

    Los booleanos representan valores de verdadero (true) o falso (false). Son útiles para estructuras condicionales.

    Ejemplo:

    verdadero = true
    falso = false

    Condicionales y Ciclos

    Los condicionales permiten tomar decisiones basadas en condiciones, mientras que los ciclos repiten instrucciones varias veces. Ambos son fundamentales para el control del flujo en los programas.

    Ciclos

    Sirven para repetir instrucciones un cierto número de veces.
