Ejemplos y Ejercicios

Aprende Python de manera didáctica con ejemplos resueltos y ejercicios interactivos. ¡Pon a prueba tus habilidades!

Ejemplo 1: Comprobación de Números Pares

Solución: Este programa determina si un número es par o impar. Solicita al usuario un número y utiliza el operador módulo (%) para verificar si el residuo al dividir entre 2 es cero. Si es así, el número es par; de lo contrario, es impar.


# Solicitamos un número al usuario
numero = int(input("Introduce un número: "))

# Verificamos si es par o impar
if numero % 2 == 0:
    print(f"El número {numero} es par.")
else:
    print(f"El número {numero} es impar.")

Ejemplo 2: Calculadora de Promedio

Solución: Este programa calcula el promedio de una lista de números proporcionados por el usuario. Utiliza split() para dividir los números ingresados como texto, los convierte en enteros y luego calcula el promedio usando sum() y len().


# Solicitamos números al usuario separados por comas
numeros = input("Introduce números separados por comas: ")

# Convertimos la cadena a una lista de enteros
numeros_lista = [int(n) for n in numeros.split(",")]

# Calculamos el promedio
promedio = sum(numeros_lista) / len(numeros_lista)

print(f"El promedio es: {promedio}")

Ejercicio 1: Suma de Tres Números

Problema: Escribe un programa que solicite tres números al usuario y muestre su suma.


# Paso 1: Solicita tres números al usuario
num1 = int(input("Introduce el primer número: "))
num2 = int(input("Introduce el segundo número: "))
num3 = int(input("Introduce el tercer número: "))

# Paso 2: Calcula la suma
suma = num1 + num2 + num3

# Resultado: Muestra la suma
print(f"La suma de los números es: {suma}")

Ejercicio 2: Área de un Triángulo

Problema: Crea un programa que calcule el área de un triángulo dados la base y la altura.


# Paso 1: Solicita la base y la altura al usuario
base = float(input("Introduce la base del triángulo: "))
altura = float(input("Introduce la altura del triángulo: "))

# Paso 2: Calcula el área
area = (base * altura) / 2

# Resultado: Muestra el área
print(f"El área del triángulo es: {area}")
